
# Part 2: NoSQL (MongoDB)

## Objective
This part demonstrates the use of MongoDB for handling semi-structured product data and performing NoSQL operations.

## Files in this Folder
- `products_catalog.json` – Product data stored in JSON format
- `mongodb_operations.js` – MongoDB insert, query, update, and delete operations
- `nosql_analysis.md` – Comparison of SQL vs NoSQL and use-case explanation

## How to Run
1. Ensure MongoDB service is running
2. Open MongoDB shell
3. Execute the operations file:
   mongosh < mongodb_operations.js
